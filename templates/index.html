Quick Start Guide

  1. Upload Documents

    Start by uploading your documents using the upload button in the top right corner.

  2. Explore Paragraphs

    Navigate to the Paragraphs section to view and organize extracted content.

  3. Run Similarity Analysis

    Go to Similarity Analysis to identify duplicate and similar content.

  4. Create Clusters

    Group similar paragraphs into clusters for better organization.

Pro Tip

For best results, analyze documents in batches of related content. This helps identify patterns within specific document categories.
